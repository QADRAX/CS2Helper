# @cs2helper/gsi-processor

A package for processing Counter-Strike 2 Game State Integration (GSI) data.

## Description

This package is part of the `CS2Helper` monorepo. It provides utilities and logic for parsing, validating, and handling GSI payloads from Counter-Strike 2.

## Installation

```bash
npm install @cs2helper/gsi-processor
```

## Scripts

- `npm run build`: Compiles the TypeScript code to JavaScript.
- `npm run test`: Runs the test suite using Vitest.
- `npm run lint`: Lints the codebase using ESLint.
- `npm run clean`: Removes the `dist` directory.
