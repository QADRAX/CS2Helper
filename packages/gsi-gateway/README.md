# @cs2helper/gsi-gateway

A package that acts as the entry point or server for receiving Counter-Strike 2 Game State Integration (GSI) data payloads.

## Description

This package is part of the `CS2Helper` monorepo. It serves as the gateway to capture incoming HTTP requests containing GSI data from CS2, and processes them using the internal `@cs2helper/gsi-processor` package.

## Installation

```bash
npm install @cs2helper/gsi-gateway
```

## Scripts

- `npm run build`: Compiles the TypeScript code to JavaScript.
- `npm run test`: Runs the test suite using Vitest.
- `npm run lint`: Lints the codebase using ESLint.
- `npm run clean`: Removes the `dist` directory.
